# Persisting and rehydrating ag-grid state with angular

Working repository for medium article [Persisting and rehydrating ag-grid state with angular](https://medium.com/@daveahern/persisting-and-rehydrating-ag-grid-state-with-angular-3f48cfedbcc8)

## Prerequisites

Install dependencies

```bash
pnpm i
```

## Run the app

```bash
pnpm run start
```
